# Multi-Page Navigation: Building a Professional Website

Transform your single-page portfolio into a complete multi-page website with professional navigation, just like the websites you visit every day.

In this lesson, you'll learn how to organize multiple HTML pages, create navigation menus, and link pages together. By the end, your portfolio will have separate About, Projects, and Contact pages with smooth navigation between them!

## Understanding Multi-Page Websites

Most professional websites aren't just one long page - they're organized into multiple pages that serve different purposes. Think about websites you use daily: they have Home pages, About pages, Contact pages, and more.

![Multi-Page Website Structure|800x600](/images/lessons/html-css/multi-page-structure.{scheme}.svg)

### Why Multi-Page Websites Matter

**Organization**: Different content gets its own dedicated space
**User Experience**: Visitors can find specific information quickly  
**Professional Appearance**: Looks like real business and portfolio websites
**SEO Benefits**: Search engines can index different pages for different topics

## Creating Your Navigation System

Navigation is how visitors move between your pages. Let's build a professional navigation menu that works on all your pages.

### Step 1: Design Your Navigation Menu

```html
<!-- Professional Navigation Component -->
<nav class="main-navigation">
    <div class="nav-container">
        <!-- Your site logo/name -->
        <div class="nav-logo">
            <a href="index.html">Alex Chen</a>
        </div>
        
        <!-- Navigation menu -->
        <ul class="nav-menu">
            <li class="nav-item">
                <a href="index.html" class="nav-link">Home</a>
            </li>
            <li class="nav-item">
                <a href="about.html" class="nav-link">About</a>
            </li>
            <li class="nav-item">
                <a href="projects.html" class="nav-link">Projects</a>
            </li>
            <li class="nav-item">
                <a href="contact.html" class="nav-link">Contact</a>
            </li>
        </ul>
    </div>
</nav>
```

### Step 2: Style Your Navigation

