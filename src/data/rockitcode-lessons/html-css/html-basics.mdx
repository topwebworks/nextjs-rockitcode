# Your First HTML Page

Welcome to your first HTML lesson! In this lesson, you'll learn how to create your very first web page using HTML.

## What You'll Learn

- What HTML is and why it's important
- How to create basic HTML structure
- How to add text content to web pages
- How to view your HTML page in a browser

## Getting Started

HTML (HyperText Markup Language) is the standard language for creating web pages. Think of it as the skeleton of every website you see!

Let's start by creating a simple "Hello World" web page.

## Your First Challenge

Try creating your first HTML page with a heading and paragraph!
